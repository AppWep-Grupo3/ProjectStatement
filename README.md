## 3.4. Product Backlog.
| User Story Id | Título                                      | Descripción                                                                                                     | Story Points |
|---------------|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|--------------|
| 1             | US01 - Registrarse                         | Como usuario quiero poder registrarme en la aplicación web para poder acceder a sus funciones y servicios.     | 3            |
| 2             | US02 - Iniciar sesión                      | Como usuario con una cuenta activa, quiero poder iniciar sesión en la aplicación web para acceder a sus funciones y servicios. | 2 |
| 3             | US03 - Visualizar suscripciones            | Como visitante, quiero ver las suscripciones disponibles para conocer los beneficios de estas. | 1 |
| 4             | US04 - Gestión de suscripciones            | Como usuario, quiero poder elegir entre una suscripción gratuita o premium para acceder a diferentes beneficios. | 2 |
| 5             | US05 - Explorar los componentes y/o periféricos disponibles | Como usuario, quiero ver una lista de componentes y periféricos disponibles para la compra de alguno. | 3 |
| 6             | US06 - Detalles del producto               | Como usuario quiero poder ver los detalles y características de un producto para decidir comprarlo. | 2 |
| 7             | US07 - Filtrar componentes y/o periféricos  | Como usuario, quiero poder filtrar los productos por características (categoría, marca, etc) que me interesan para decidir comprarlo. | 2 |
| 8             | US08 - Seleccionar un producto para la compra | Como usuario quiero poder elegir un producto para realizar la compra de este. | 3 |
| 9             | US09 - Subir imágenes de producto usado     | Como usuario, quiero poder subir imágenes del producto usado para darlo como parte de pago. | 2 |
| 10            | US10 - Cotización automática de producto usado para descuento | Como usuario premium quiero que el descuento aproximado de mi nuevo producto se calcule automáticamente según los productos usados que elija para manejar un presupuesto anticipado | 3 |
| 11            | US11 - Visualizar el historial de compras    | Como usuario, quiero poder revisar mi historial de compras para tener un control de los productos que he comprado. | 2 |
| 12            | US12 - Actualizar perfil                     | Como usuario, quiero poder actualizar la información en mi perfil para mantener mi usuario al día. | 2 |
| 13            | US13 - Cerrar Sesión                         | Como usuario quiero poder cerrar sesión en mi cuenta para mantener la seguridad y privacidad de mi cuenta | 1 |
| 14            | US14 - Recuperar contraseña                  | Como usuario quiero poder recuperar mi contraseña para evitar la pérdida de mi cuenta asociada a la aplicación web. | 2 |
| 15            | US15 - Cambiar contraseña                    | Como usuario, quiero poder cambiar mi contraseña actual por una nueva para mantener la seguridad de mi cuenta | 2 |
| 16            | US16 - Orientaciones para contraseña segura   | Como usuario quiero que la plataforma me indique los requisitos para crear una contraseña segura para mantener la seguridad alta en mi cuenta | 1 |
| 17            | US17 - Añadir un método de pago               | Como usuario quiero poder añadir un método de pago (tarjeta de crédito/débito o cuenta bancaria) para realizar la compra de un producto rápidamente. | 2 |
| 18            | US18 - Eliminar método de pago                | Como usuario, quiero poder eliminar un método de pago ingresado en la aplicación web para mantener la seguridad de este. | 1 |
| 19            | US19 - Seleccionar método de pago             | Como usuario quiero poder seleccionar un método de pago para realizar una compra. | 2 |
| 20            | US20 - Acerca de nosotros                     | Como visitante, quiero conocer la rama profesional del equipo de desarrollo | 1 |
| 21            | US21 - Probar las funciones del aplicativo web | Como visitante quiero tener un acercamiento al funcionamiento del aplicativo web | 1 |
| 22            | US22 - Registro e inicio de sesión desde la landing page | Como visitante quiero registrarme o iniciar sesión, una vez inspeccionado el landing page para probar las funcionalidades del aplicativo web | 2 |
